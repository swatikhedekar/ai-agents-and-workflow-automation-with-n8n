# Automated Email Notification on Form Submission Using n8n 
## 1. Project Overview 
This project demonstrates how to automate sent email notifications using n8n form trigger 
whenever a user submits a form. The automation captures form data, processes it, and sends 
a confirmation or notification email automatically without manual intervention. This workflow 
helps reduce manual work, improves response time, and ensures consistent communication 
with users or internal teams. 
## 2. Objective 
- Automate email notifications on form submission 
- Learn n8n workflow structure and triggers 
- Integrate n8n with Google Sheets and Gmail 
- To gain hands-on experience with triggers, data handling, email automation
- n8n integration with Gmail and google sheet.
